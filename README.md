### Запуск
1. ```git clone``` -> folder
2. ```cd``` folder
3. ```npm i```
4. ```npm run``` 
	+ ```start``` (запустит сборку статики и сервер)
	+ ```server``` (запустит только сервер)
- Команда ```gulp``` (встроена в ```npm run start```) отдельно произведет билд статических ресурсов в папку ```build```
